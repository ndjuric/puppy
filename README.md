# puppy
PHP boilerplate project developed as part of an interview assignment wherein one of the conditions was not to use frameworks or any third party libraries on the backend.

# Run
```bash
~/works $ sudo apt install docker-compose git
~/works $ git clone https://github.com/ndjuric/puppy.git
~/works $ cd puppy/
~/works/puppy$ docker-compose up  
```
Try opening http://localhost/api/test_endpoint from your browser.
